🌍 Tourfinco – AI Powered Budget Travel Planner
🚀 Overview

Tourfinco is an AI-powered travel planning platform designed specifically for middle-class and budget travelers in India.
It helps users plan complete trips based on their budget using intelligent recommendations and a community-driven marketplace of local vendors.

The goal is simple:
👉 Make travel affordable, accessible, and smart.

✨ Features
🤖 AI Travel Planner
Generates complete travel itineraries
Based on:
Destination
Number of travelers
Budget
Trip duration
Includes:
Places to visit
Food recommendations
Accommodation suggestions
Transportation planning
🛒 Vendor Marketplace (Meesho Model)
Connects users with local vendors across India
Vendors include:
Hotels
Local guides
Transport providers
Food services
Small shops
Focus on low-cost, budget-friendly services
🗺️ Google Maps Integration
View destinations and routes
Explore nearby vendors
Navigation and travel time estimation
🔐 Authentication System
Email & Password login
OTP-based authentication
Secure session handling
📊 Budget Optimization
Cost breakdown (stay, food, travel, activities)
Smart budget allocation
Cheapest route suggestions
📁 User Features
Save trips
Download itinerary as PDF
Share trip plans
View past trips
Wishlist destinations
🧑‍💼 Admin Panel
Manage users and vendors
Approve/reject vendor listings
Track revenue and commissions
Monitor platform activity
Control AI responses and pricing logic
🧠 How It Works
User logs in
Answers 4 simple questions:
Destination
Number of travelers
Budget
Duration
AI generates a complete travel plan
User explores and books affordable vendors
💰 Business Model

Tourfinco generates revenue through:

5% commission on bookings
Vendor subscriptions for premium listings
Advertisements for vendors
Premium user plans
🛠️ Tech Stack
Frontend:
React
Tailwind CSS
Framer Motion
Three.js
Backend:
Supabase (Auth, Database, Storage)
APIs:
Google Gemini AI (Trip generation)
Google Maps API (Location & navigation)
🗄️ Database Structure (Supabase)

Main Tables:

users
trips
vendors
bookings
reviews
🔐 Security
API keys stored securely (Supabase / environment variables)
Row Level Security (RLS) enabled
No sensitive data exposed on frontend
